## Preconditions

- MPC_002 — CardioCheck app accessible at https://cardiocheck-releasev140.up.railway.app
- MPC_003 — User is logged in and on the dashboard

2. Log in and wait for dashboard to fully load
3. Run `axe.run()` on the dashboard page
4. Collect all violations
5. Log each violation with: description, impact, element, and help URL
